## Design Context

### Users
Dueños de PyMEs y comercios locales en Mendoza, Argentina (restaurantes, bodegas, peluquerías, consultorios, alojamientos). Generalmente +35 años, no muy técnicos, buscan presencia digital profesional a precio accesible. Llegan desde WhatsApp, Instagram, LinkedIn o boca en boca. El job-to-be-done: decidir rápido si este desarrollador es confiable y contratar una web para su negocio.

### Brand Personality
Profesional, cercano, técnico-pero-accesible. Un ingeniero de software real (no agencia) que habla claro y entrega calidad a precio justo. Transmitir confianza técnica sin intimidar.

### Aesthetic Direction
- **Tono visual**: Dark mode tech-profesional con acentos azules. Inspiración Apple/Vercel pero adaptado a público argentino no-tech.
- **Tema**: Dark (#0a0f1c fondo) con accent azul (#3b82f6) y toques violeta/teal como secundarios.
- **Tipografía**: system-ui stack (sin Google Fonts externas por performance).
- **Elementos decorativos**: Orbes animados sutiles, grid de fondo con fade radial, code mockup como elemento ambiental.
- **Anti-referencias**: No parecer plantilla genérica de Wix/WordPress. No parecer agencia cara. No parecer amateur.
- **Glass morphism**: Usado intencionalmente en cards y formulario para reforzar el look tech, no como decoración gratuita.

### Design Principles
1. **Confianza inmediata**: Cada elemento debe reforzar credibilidad (experiencia en Globant, demos reales, prototipo gratis).
2. **Velocidad de decisión**: El visitante debe entender la propuesta y poder actuar en <30 segundos (CTA claros, precios visibles, WhatsApp directo).
3. **Performance primero**: Sitio single-file, sin dependencias externas, carga instantánea — el sitio mismo es la demo de calidad.
4. **Accesible para no-técnicos**: Lenguaje claro, navegación obvia, formulario guiado paso a paso.
5. **Localización**: Español rioplatense, precios en ARS, referencias a Mendoza, tono cercano pero profesional.
